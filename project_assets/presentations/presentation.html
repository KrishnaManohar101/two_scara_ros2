Forward Kinematics (FK)

Mapping Joint Angles to 3D Space

// 2D Revolute Planar Projection
x = L₁ cos(θ₁) + L₂ cos(θ₁ + θ₂)
y = L₁ sin(θ₁) + L₂ sin(θ₁ + θ₂)
// Prismatic Vertical Joint
z = H_base - d₃

The FK calculation uses trigonometric vector addition to determine the precise location of the end-effector based on the current state of all three joints.

Inverse Kinematics (IK)

The Geometric Solution (Law of Cosines)

// Joint 2 Angle
θ₂ = cos⁻¹((x² + y² - L₁² - L₂²) / 2L₁L₂)
// Joint 1 Angle
θ₁ = atan2(y, x) - atan2(L₂ sin θ₂, L₁ + L₂ cos θ₂)

Our solver handles both 'Elbow-Up' and 'Elbow-Down' solutions and rejects coordinates outside the toroidal workspace (R_max = 2.5m).

Robot Dynamics

Lagrangian Physics & Inertia

// Mass Matrix Element (Inertia)
M₁₁ = (m₁+m₂+mₚ)L₁² + (m₂+mₚ)(L₂² + 2L₁L₂ cos θ₂)
// Vertical Gravity Compensation
G₃ = (mass_link + mass_payload) * g

By modeling the mass of link segments and variable payloads, the system calculates the exact required torques for smooth, physics-aware motion.

Trajectory Optimization

Applying Unit-1: Lagrange Multipliers

Problem: Move in a perfect straight line while minimizing joint velocities.

Minimize: f(θ̇) = ½ θ̇ᵀ θ̇
Subject to: g(θ̇) = J(θ)θ̇ - v_target = 0

The Lagrangian Solution yields the Control Law:

θ̇* = Jᵀ(JJᵀ)⁻¹ v_target = J⁺ v_target

This ensures Straight-Line Motion, crucial for avoid conveyor collisions.
